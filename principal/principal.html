Preguntas Frecuentes

¿Es necesario un hardware específico para operar La Perla?

No, cualquier computadora que cumpla con los requisitos mínimos mencionados será suficiente para operar La Perla. No es necesario invertir en hardware costoso o en un kit punto específico.

¿Qué es La Perla?

La Perla es un sistema de inventario diseñado para gestionar de manera eficiente las ventas, los productos y el stock en una tienda. Permite registrar ventas, administrar productos y generar reportes, todo desde una interfaz sencilla e intuitiva.

¿Puedo generar facturas con La Perla?

Sí, La Perla permite generar facturas en formato PDF directamente desde la base de datos, incluyendo detalles de la venta, el nombre de la tienda, y un diseño personalizado.

¿Cómo se realiza la recuperación de contraseñas en La Perla?

La Perla cuenta con un proceso de recuperación de contraseñas que envía un código de 6 dígitos al correo del usuario. Este código se almacena en la base de datos y se valida para permitir el cambio de contraseña.

¿Es posible modificar el inventario desde La Perla?

Sí, el sistema permite actualizar el inventario, añadiendo nuevos productos, modificando la cantidad de stock disponible y eliminando productos que ya no se venden.
